# HortoCeras
Projeto de Modelagem de Banco de Dados - Cruzeiro do Sul 

# O que há neste repositório?

* README.md: Resumidamente a documentação do projeto.
* schema.sql: DDL - Script que cria a estrutura do banco e também define todas as chaves primárias e estrangeiras.
* insert.sql: DML - Script que popula o banco de dados, inserindo os dados de exemplo (clientes, produtos e pedidos).
* consultas.sql: DQL - Contém consultas com SELECT para a análise de dados.
* manipulacao.sql: DML - Contém os comandos UPDATE e DELETE por integridade.

# Como testar
É necessário executar os arquivos listados na ordem acima para que ocorra de forma correta.
